# Contributing

Thanks for your interest in contributing to 777 raid bot.

## Guidelines
- Keep changes focused and minimal.
- Test your changes before submitting.
- Do not remove safety checks or member requirements without discussion.

## Development

1. Fork the repo and create a feature branch.
2. Install dependencies with `uv sync`.
3. Make your changes.
4. Run the bot locally and verify behavior.
5. Submit a pull request with a clear description.
