## Setup

### uv (recommended)

```bash
uv sync
uv run python main.py
```

### pip

```bash
python3 -m pip install -r requirements.txt
python3 main.py
```

Then copy `config.example.json` to `config.json` and edit `config.json`, `nuke.jsonc`, and `messages.jsonc` with your settings.
